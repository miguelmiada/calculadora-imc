Calculadora de IMC 🏋️

Programa em C que calcula o Índice de Massa Corporal (IMC) do usuário e exibe a classificação correspondente.

=====================================================================

📋 Como funciona
O usuário informa seu peso (kg) e altura (m), e o programa calcula o IMC usando a fórmula:
IMC = peso / (altura × altura)

📊 Classificações
IMCClassificaçãoAbaixo de 18.5Magreza18.5 – 24.9Normal25.0 – 29.9Sobrepeso30.0 – 39.9ObesidadeAcima de 40.0Obesidade Grave
